# makeweb

MakeWeb: Make small web apps - without the tears.

## Development

### Setup

Clone makeweb repository

```console
git clone https://github.com/zentropi/makeweb.git
cd makeweb
```

Install dependencies

```console
sh tools/setup.sh
```

### Build

```console
sh tools/build.sh
```

### Run

```console
./makeweb main.py
```
